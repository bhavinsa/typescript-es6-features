### ES6 Features

 - let, const & enum
 - string interpolation
 - optional parameter
 - Interface
 - Class
 - Arrow Function
-  GET & SET Properties
-  Module
-  Spread operator
-  Rest operator
-  Generator
-  Destructuring

```    
npm i -g typescript
git clone https://github.com/bhavinsa/typescript-es6-features.git
cd typescript-es6-features
tsc --target es5 main.ts && node main.js

```
